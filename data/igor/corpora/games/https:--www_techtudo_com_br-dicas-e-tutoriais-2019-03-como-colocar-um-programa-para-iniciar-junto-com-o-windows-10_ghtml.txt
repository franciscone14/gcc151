Como colocar um programa para iniciar junto com o Windows 10Dica mostra como adicionar programas na inicialização do Windows 10 para que eles abram assim que o sistema estiver pronto para uso.Por Marvin Costa, para o TechTudo10/03/2019 07h00 Atualizado 2019-03-10T10:00:53.187ZO Windows 10 permite que usuários escolham programas para iniciar automaticamente toda vez que o computador for ligado. Ideal para pessoas que usam o PC para trabalho e atividades acadêmicas, o procedimento é um truque rápido e alternativo no menu Iniciar que não necessita de alterações nas configurações do sistema. A ação garante que seus programas preferidos sejam iniciados assim que você ligar o computador com Windows 10. Para saber como adicionar um programa na inicialização do Windows 10, confira o passo a passo a seguir.Ação para abrir o menu iniciar do Windows 10 para verificar os programas que podem entrar na inicialização do sistema — Foto: Reprodução/Marvin CostaPasso 2. Clique com o botão direito do mouse sobre o software que deseja adicionar na inicialização do sistema. Em seguida, repouse o ponteiro do mouse sobre a opção "Mais";Ação para verificar opções para um programa no menu iniciar do Windows 10 — Foto: Reprodução/Marvin CostaPasso 3. Para prosseguir, clique na opção "Abrir local do Arquivo", como mostra a imagem abaixo;Ação para abrir o local de um software no Windows 10 — Foto: Reprodução/Marvin CostaPasso 4. Com local do arquivo aberto, use a combinação de teclas "Windows" e "R". Nesse momento, digite o comando "shell:startup" e clique no botão "OK". Essa ação abrirá a pasta de inicialização do sistema.
