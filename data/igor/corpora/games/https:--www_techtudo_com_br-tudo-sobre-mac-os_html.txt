Mac OS XMac OS X é o mais avançado sistema da Apple; conheçaMac OS X foi o sistema operacional oficial dos computadores Mac, da Apple. Inicialmente, o sistema foi criado com base no NeXTSTEP, que por sua vez, era um sistema operacional Unix baseado no Mach, mais com código fonte do Unix BSD.O NeXTSTEP era da companhia NeXT, a empresa criada por Steve Jobs. Foi adquirida pela Apple para trazê-lo de volta e fornecer a estrutura que a gigante de Cupertino precisava para substituir a envelhecida plataforma Mac OS Classic.Mac OS X 10.0 Cheetah: Lançado em 24 de março de 2001, o Mac OS X 10 Cheetah foi a primeira versão do sistema Mac OS X. A interface do sistema foi revolucionária para a época. O sistema tinha requisitos de hardware altos, exigindo um Mac com pelo menos um processador G3 e 128 MB de RAM.Mac OS X 10.1 Puma: A Apple lançou o Mac OS X 10.1 Puma poucos meses depois do lançamento do Cheetah, em 25 de setembro de 2001, para resolver muitos problemas da versão original. Essa versão, oferecida gratuitamente para os proprietários do Cheetah, vinha com suporte para mais impressoras, gravava CD e DVD e melhorou o desempenho de gráficos 3D nas telas da época.Mac OS X 10.2 Jaguar: O Mac OS X 10.2 Jaguar foi lançado em 23 de agosto de 2002. A partir dessa versão 10.2, a Apple tornou o OS X o sistema operacional de inicialização padrão de seus equipamentos.Mac OS X 10.3 Panther: Lançado em 24 de outubro de 2003, o OS X 10.3 Panther elevou os requisitos do sistema em comparação com as versões anteriores do sistema operacional, removeu o Internet Explorer e marcou o aparecimento do Safari como navegador padrão do OS X. Também acrescentou uma nova aparência em alumínio escovado para as janelas abertas e incluiu um aplicativo iChat renovado, que agora permitia chats de vídeo em tempo real em computadores com câmeras FireWire e conexões de Internet com banda larga.Mac OS X 10.4 Tiger: Tiger vinha com o utilitário de busca Spotlight e o Dashboard, uma área que exibia mini-aplicativos que a Apple chamava de Widgets.Mac OS X 10.5 Leopard: A Apple lançou em 26 de outubro de 2007 o Mac OS X 10.5 Leopard. O recurso de backup Time Machine foi uma das novas características mais notáveis do Leopard. Essa versão deu uma aparência nova ao ambiente de trabalho do Mac, mais elegante com um dock redesenhado e um novo papel de parede padrão. Essa, também foi a versão que finalizou o suporte do Mac OS X para a linha de Macs antigos baseados em processadores PowerPC.Mac OS X 10.6 Snow Leopard: O Mac OS X 10.6 Snow Leopard marcou outro grande aumento das necessidades de sistema, removendo de vez o suporte para processadores PowerPC, incluindo suporte para processadores de 64 bits, exigindo uma máquina com pelo menos 1 GB de RAM e consumindo menos 250 MB de espaço em disco. Nessa versão, a Apple passou a cobrar um valor baixo pelos sistema (apenas U$$ 29), para acelerar a adoção do mesmo.OS X 10.7 Lion: Anunciado em 20 de Outubro de 2010 e lançado em 20 de Julho de 2011, o Mac OS X Lion trouxe o Mission Control, uma nova maneira de ver todas as janelas abertas reunidas de acordo com programas de agrupamento.OS X 10.8 Mountain Lion: O sucessor do Lion foi anunciado em Fevereiro de 2012, lançado em 25 de Julho de 2012, disponibilizado para download na Mac App Store, e a primeira versão a ser chamada oficialmente de “OS X”, em vez de “Mac OS X”. Suas maiores atualizações foram a introdução do iCloud, Messages, Reminders, Notes, Notification Center, Game Center, dentre outras aplicações vindas do iOS.OS X 10.9 Mavericks: Mavericks foi a primeira versão do OS X que não usava o nome de algum felino. O sistema foi anunciado em Junho de 2013, lançado em 22 de outubro de 2013 e suas principais atualizações foram os aplicativos iBooks e Mapas, que antes só estavam disponíveis para o iOS.OS X 10.10 Yosemite: Seguindo a mudança no esquema de nomenclatura apresentado com o OS X Mavericks, o OS X 10.10 recebeu o nome um parque nacional dos Estados Unidos, localizado no estado da Califórnia, (Yosemite National Park). Ele foi anunciado em junho de 2014 e lançado no dia 16 de outubro do mesmo ano. Essa versão trouxe uma central de notificações completa e suporte ao recurso handoff (que permite continuar trabalhos iniciados no iPhone, iPad ou outro Mac).Mac OS X 10.11 El Capitan: Trazendo poucas inovações em relação ao Yosemite, o principal destaque do El Capitan é o “Split View”, um recurso que permite preencher a tela do Mac com dois aplicativos que o usuário escolher, lado a lado, de forma fácil. Ainda assim, esse versão trouxe diversos ajustes, novas ferramentas e melhoras nos aspectos do sistema anteriorNossa opiniãoO Mac OS X é um sistema moderno, fácil de usar e com uma aparência elegante. Juntando isso com a afinidade do sistema com o hardware da Apple, obtêm-se uma ótima experiência de uso.Atualmente, o Mac OS X é oferecido gratuitamente através da loja de aplicativos da Apple. Isso certamente ajuda a aumentar a adesão dos usuários e a fazer com que eles atualizem seus equipamentos.Outro ponto de destaque do sistema da Apple é a sua integração e sintonia com o sistema mobile da empresa. Essa característica entrega ao usuário um certo conforto, o que muitas vezes, o leva a comprar outros equipamentos da marca.Como o Mac OS X é um sistema extremamente amarrado ao hardware da Apple, não pode ser instalado em outro equipamento (nem sequer é suportado nessa situação).
